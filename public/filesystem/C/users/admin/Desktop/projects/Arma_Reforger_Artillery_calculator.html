Arma Reforger Artillery Calculator

This is a web-based artillery calculator built for Arma Reforger. It allows users to select artillery type, shell type, origin point, and target point, then calculates elevation, charges, azimuth, and time of flight based on detailed range data.

Features include:

  • Support for US and RU artillery systems
  • Dynamic map interaction with custom maps
  • Ring-based firing data, multiple shell types, and interpolation
  • Visual range indicators and trajectory information
