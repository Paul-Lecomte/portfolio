📦 Stocker – Inventory Management System

About the Project

Stocker is a full-stack inventory management system designed to simplify how warehouses, small businesses, or retailers manage their stock in real time.

With a user-friendly interface and robust backend, it allows seamless product tracking, movement logging, and dynamic stock updates.

Key Features

  • ✔️ User Authentication (JWT-based)
  • ✔️ Role-based Access for Admin and Users
  • ✔️ Add, Update, and Remove Inventory Items
  • ✔️ Real-time Quantity Updates
  • ✔️ Product Category Management
  • ✔️ RESTful API Integration
