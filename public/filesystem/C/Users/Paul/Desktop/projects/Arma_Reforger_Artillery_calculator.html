🎯 Arma Reforger Artillery Calculator

About the Project

The Arma Reforger Artillery Calculator is a web-based tactical tool built to assist players and mission designers in Arma Reforger. It streamlines the process of calculating artillery firing solutions based on realistic physics, accurate firing tables, and custom user inputs.

The app supports selecting artillery and shell types, marking origin and target positions directly on a map, and outputs precise firing data such as charge, elevation, azimuth, and time of flight using real ballistic characteristics.

Key Features

  • 🔄 Dynamic and interactive custom map UI
  • 🇺🇸🇷🇺 Supports both US and Soviet-style artillery systems
  • 💥 Multiple shell types: HE, Smoke, Illumination, and Practice
  • 📏 Accurate interpolation between ring data points
  • 🎯 Displays calculated azimuth, elevation, charge level, and impact timing
  • 📡 Visual range indicators and distance overlays on the map

Why I Built It

I wanted to build a practical military simulation tool that enhances the tactical experience in Arma Reforger.
